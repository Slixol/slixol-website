# 🏭 Page Factory

Welcome to **Page Factory** — the ultimate engine for generating landing pages fast. 

Built on a highly scalable [Nx](https://nx.dev/) monorepo architecture and powered by [Next.js](https://nextjs.org/), Page Factory provides a streamlined workflow for marketing and engineering teams to spin up beautiful, high-performing landing pages in record time. 

### ✨ Why Page Factory?
* **Lightning Fast:** Generate new pages instantly using shared Next.js templates.
* **Modular Architecture:** Thanks to Nx, all UI components, utilities, and styling are shared across different landing page apps, ensuring brand consistency and DRY code.
* **Blazing Build Times:** Leverage Nx's computation caching and affected builds to only test and deploy the pages that actually changed.
* **SEO Optimized:** Built with Next.js App Router for top-tier performance, Core Web Vitals, and server-side rendering out of the box.
